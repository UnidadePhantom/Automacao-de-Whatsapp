# Bot de WhatsApp para Automação de Vendas e Atendimento

![Node.js](https://img.shields.io/badge/Automação-Node.js-green?logo=node.js)
## 📖 Descrição
Projeto desenvolvido em **Node.js** para automação de processos de vendas e atendimento via WhatsApp.  
Este bot foi criado como demonstração de habilidades técnicas em integração de APIs, automação de mensagens e engenharia comercial aplicada.

Item complementar do meu [portfólio](https://github.com/UnidadePhantom/Portf-lio.git) para fins de apresentação profissional.

## 🛠️ Tecnologias
- Linguagem: Node.js (JavaScript)
- Integração: WhatsApp Business API / Baileys
- Paradigma: Automação de processos e comunicação digital

## 🎯 Objetivo
- Automatizar respostas a clientes em tempo real.  
- Suportar processos de vendas e pós-venda com eficiência.  
- Demonstrar aplicação prática de **engenharia de sistemas digitais** em contexto comercial.
